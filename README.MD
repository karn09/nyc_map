# Neighborhood Map Project
Goal: In this project I used the Google Maps API to enable me to search restaurants local to NYC and return inspection letter grades from the NYC Open Data API. 

More information about NYC inspection letter grading system can be found here:
http://www.ehagroup.com/food-safety/new-york-abc-restaurant-grading/

Technologies used:
* Google Maps API
* Knockout
* jQuery
* NYC Open Data API

In order to run, just open index.html
